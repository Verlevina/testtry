# Тестовый проект для проекта ВП "АНАК"

Включает  в  себя 3 библиотеки классов 
 *  Helper  - содердит в себе обертки над  Selenium и вспомогательные классы
 *  Pages - содержит описание элементов и стандартных действий на странице, повторяет иерархическую структуру url ВП "Анак"
 *  VPAnak - содержит тестовый NUnit проект
